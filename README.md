# Конструктор актов

Статический сервис для формирования актов сдачи-приемки оказанных услуг для договоров Центрального офиса Русского Регистра.

## v34

- Выгрузка результата в редактируемый XLSX.
- Предварительный просмотр приведен к визуальной структуре Excel-шаблона: сетка, подписи строк/столбцов, блоки и расположение полей соответствуют итоговому XLSX.
- Локальные шрифты Manrope.
- Без backend, CDN и внешних запросов.

## Публикация

Загрузите содержимое архива в репозиторий GitHub Pages.
