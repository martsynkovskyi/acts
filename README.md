# Конструктор актов сдачи-приемки

Статический сервис для договоров Центрального офиса Русского Регистра.

## Что внутри

- `index.html` - основной файл сервиса
- `favicon.svg` - иконка сайта
- `preview.svg` - изображение для OpenGraph-превью
- `CNAME` - домен `acts.martsynkovskyi.ru`

## Возможности

- заполнение реквизитов акта
- предпросмотр печатной формы
- расчет НДС
- сумма прописью
- сохранение данных в браузере через localStorage
- скачивание редактируемого `.docx` без backend, CDN и внешних зависимостей

## Публикация на GitHub Pages

Загрузите содержимое архива в репозиторий GitHub Pages. Файл `CNAME` уже подготовлен для домена `acts.martsynkovskyi.ru`.


Версия 1.1 (31.05.2026)


## Шрифты
Интерфейс использует локально подключенный Manrope из папки `fonts`, без CDN и внешних запросов.
