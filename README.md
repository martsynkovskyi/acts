# Генератор актов сдачи-приемки

Статический web-сервис для генерации актов сдачи-приемки оказанных услуг.

## Состав

- `index.html` — весь интерфейс, логика расчета, предпросмотр и экспорт PDF.
- `favicon.svg` — иконка сайта.
- `preview.svg` — OpenGraph preview для мессенджеров.
- `CNAME` — домен GitHub Pages: `acts.martsynkovskyi.ru`.

## Особенности

- Чистый HTML/CSS/JavaScript.
- Без backend, CDN и внешних зависимостей.
- Прямое скачивание PDF без `window.print()`.
- Данные остаются в браузере пользователя.
- Поддержка Ассоциации Русский Регистр, ООО «РР МС», ООО «РРС».
- Автоматический расчет НДС и сумма прописью.
